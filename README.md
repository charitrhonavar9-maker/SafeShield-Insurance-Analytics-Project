# 🛡️ SafeShield Insurance Analytics Dashboard

## 📌 Project Overview

The **SafeShield Insurance Analytics Dashboard** is an end-to-end Business Intelligence project developed using **MySQL, SQL, Power BI, DAX, and Excel**. The project analyzes insurance operations, including policy performance, claims, fraud detection, customer behavior, branch performance, and financial insights through interactive dashboards.

This project demonstrates the complete data analytics workflow, from raw data analysis to executive-level dashboards.

---

## 🎯 Business Problem

Insurance companies manage large volumes of policy, claim, customer, and payment data. The objective of this project is to transform raw insurance data into actionable insights that help business leaders:

- Monitor premium collection
- Analyze claim trends
- Detect fraud patterns
- Evaluate branch performance
- Understand customer behavior
- Measure financial performance
- Support data-driven decision making
